# Example job recipes

Hand-written Harbor job YAMLs for **local smoke** and demos. One persona per job (`persona_0042`) unless noted.

```bash
# No API key
uv run harbor run -c configs/jobs/example-job-recipe/harbor-smoke-local.yaml

# Persona survey (needs ANTHROPIC_API_KEY)
uv run harbor run -c configs/jobs/example-job-recipe/appSim-example-survey-local.yaml
```

These are **not** generated by scripts. For multi-persona batches, see [getting-started.md](../../docs/applications/getting-started.md#7-batch--sample-many-personas-job) (output under `configs/jobs/application-task-job-recipe/`, gitignored).
