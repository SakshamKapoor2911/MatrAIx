# Synthetic Persona Generation Survey

Overleaf-ready starter project for a survey on synthetic persona generation for LLM agents.

## Main Files

- `main.tex`: paper entry point.
- `sections/`: empty section stubs.
- `references.bib`: generated starter bibliography from the literature corpus.
- `literature/papers_merged_top100.jsonl`: canonical screening/coding records.
- `literature/papers_merged_top100.csv`: spreadsheet-friendly index.

## Literature Workflow

The corpus is generated by:

1. `tools/collect_literature.ps1`
2. `tools/collect_arxiv.ps1`
3. `tools/merge_literature.ps1`

Search logs are preserved under `literature/`.
