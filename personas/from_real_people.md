# Getting persona data from real people e.g. Wikipedia
This is for Sub-task 2.3 🧑 Personas from real human info: can we start collecting this? [Owners: TBD @Brihi Joshi - USC @Sankalp Jajee - USC @Rosie Gao - Snap @Ishan Gupta @Tiasa Singha Roy - NYU @Yilan (Eliza) Fan - Microsoft @Xiaoyi Liu - Brown @Zonglin - UCSC]

In today (June 20, 2026) discussion, the team discussed using Wikipedia profiles, including both real and fictional characters. Instead of web-scraping, we consider using existing data people have already organized, such as on huggingface websitte

| Dataset                                |       Real people? |                Fictional people? |                     Has article text? | Notes                                                                                                                                                                                                                                                                                                                           |
| -------------------------------------- | -----------------: | -------------------------------: | ------------------------------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`michaelauli/wiki_bio`([Link][1])**             |                Yes |                        Mostly no |              Lead paragraph + infobox | Classic WikiBio (biographies), from English Wikipedia, with first paragraph and infobox. Could be nice for “persona seed” summaries. There's a paper [Neural Text Generation from Structured Data with Application to the Biography Domain](https://arxiv.org/abs/1603.07771)                                                                               |
| **`rcds/wikipedia-persons-masked`([Link][2])**    |                Yes |        likely mostly real people (?) |                 Full text + sentences | ~**70k Wikipedia pages about people**, built by querying Wikidata for people and filtering the HF Wikipedia dataset. Has original text, and a bunch of others like masking which don't seem super useful here. There's a lot of "[More Information Needed]"s ...                                                                                                        |
| **`wikimedia/structured-wikipedia`([Link][3])**   | Yes (need filter) |               Yes (need filter) |              Full structured articles | Official Wikimedia HF dataset: English/**French** Wikipedia articles in Parquet, with `main_entity` Wikidata QID, abstract, description, infoboxes, sections, references, images(i think?). It's new from **May 13, 2026**. Many downloads! |
| **`wikimedia/wikipedia`([Link][4])**              | Yes (need filter) |               Yes ( needfiltering) |                    Plain article text | Simpler full-Wikipedia HF dataset. **Many languages**                                                                                                                                                                               |
| **`Pclanglais/AllWikidataCharacters` ([Link][5])** |                 No |                              Yes |                   No, IDs/labels only | It's all **fictional** achracters. May be accessed using ways like `structured-wikipedia.main_entity.identifier`.                                                                                                                         |
| **`Vijaysr4/en_wikidata_5M_entities` ([Link][6])** | Yes (need filter) |               Yes (need filter) |               Metadata, not full text | Gives Wikidata entities with English labels/descriptions and optional English Wikipedia title/URL. Good for entity inventory / linking, but not a primary corpus.                                                                                                                                            |
| **DBpedia ([DBpedia Association][7])**                            |  Not sure | Not sure | Structured facts, sometimes abstracts | DBpedia extracts structured content from Wikimedia projects. They're doing this structured data approach where they have a "open knowledge graph" style database-like setup. Could take more effort because it's not as straightforward as huggingface|
