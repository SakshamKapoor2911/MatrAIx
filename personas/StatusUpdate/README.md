# 📌 Status Updates

Each team member keeps a personal status-update file in this folder.

## How it works

- Create **one Markdown file named after yourself**, e.g. `First-Last.md`.
- Log your progress over time, newest entry on top.
- Each entry: a **date** + a short bullet list of what you did / are doing / blockers.

See [First-Last.md](First-Last.md) for an example.
