# Scenario Context: MDN Web Docs

MDN Web Docs is the web's primary reference for HTML, CSS, JavaScript, Web APIs, and browser compatibility specifications.

Key details:
- Base URL: `https://developer.mozilla.org/`
- Navigation: Search bar, Web Technology topic landing pages (HTML, CSS, JavaScript, Web APIs).
- Document structure: Page heading (`h1`), Overview, Syntax/Parameters, Examples, Specifications, Browser Compatibility table.
