# Scenario Context: Hacker News

Hacker News is a social news website focusing on computer science, technology, startups, and data.

Key details:
- Base URL: `https://news.ycombinator.com/`
- Layout: Table view of stories with title (`.titleline > a`), score (`.score`), user submitter, age, and comment link (`a:has-text("comments")`).
