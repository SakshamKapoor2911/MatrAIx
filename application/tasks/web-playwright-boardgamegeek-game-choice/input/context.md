# Scenario Context: BoardGameGeek

BoardGameGeek (BGG) is an online resource and community for board game hobbyists.

Key details:
- Base URL: `https://boardgamegeek.com/`
- Navigation: Top Ranked (`/browse/boardgame`), Hotness list, Search.
- Game attributes: Title, BGG Rank, Geek Rating, Avg Rating, Weight (Complexity).
