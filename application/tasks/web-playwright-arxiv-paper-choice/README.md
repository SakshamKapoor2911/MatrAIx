# web-arxiv_paper-choice

Web browsing task where an agent browses arXiv (`https://arxiv.org/`) to select a research preprint.

## Persona Stratification
- `highest_education`: `Bachelor's`, `Master's`, `Doctorate`
- `decision_style`: `Analytical`, `Deliberative`

```bash
bash solution/solve.sh
```
