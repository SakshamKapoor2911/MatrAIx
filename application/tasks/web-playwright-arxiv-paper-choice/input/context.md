# Scenario Context: arXiv

arXiv is a free distribution service and open-access archive for 2+ million scholarly articles in physics, mathematics, computer science, quantitative biology, quantitative finance, statistics, electrical engineering, and economics.

Key details:
- Base URL: `https://arxiv.org/`
- Subject archives: Computer Science (cs), Physics (phys), Math (math), etc.
- Abstract page structure: Title (`h1.title`), Authors (`.authors`), Subjects (`span.primary-subject`), Abstract block (`blockquote.abstract`).
