# Scenario Context: Books to Scrape

Books to Scrape is an open e-commerce bookstore sandbox with categories, star ratings, stock availability, and UK pricing (£).

Key details:
- Base URL: `https://books.toscrape.com/`
- Navigation: Left sidebar contains genre categories (Travel, Mystery, Historical Fiction, Sequential Art, etc.).
- Product attributes: Title, Price (£), Availability, Star Rating (One to Five), UPC code.
